fn add_element(element: ElementData)

Add a new element in the elements list. The additional list is searched before the PERIODIC_TABLE array, so adding a new element here will make it overwrite the real one.

This function use a RwLock to protect itself against race conditions, and will block. As such, it should only be called by one thread, and preferably before any parallel region.
