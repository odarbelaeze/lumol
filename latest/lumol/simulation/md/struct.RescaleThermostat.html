Velocity rescaling thermostat.

The velocity rescale algorithm controls the temperature by rescaling all the velocities when the temperature differs exceedingly from the desired temperature. A tolerance parameter prevent this algorithm from running too often: if tolerance is 10K and the target temperature is 300K, the algorithm will only run if the instant temperature is below 290K or above 310K.
