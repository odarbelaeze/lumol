pub struct Vector3D(_);

A simple 3-dimensional vector type, storing three f64.

A Vector3D implement all the arithmetics operations: ``rust //u,vandware vectors.aandb` are scalar let u = Vector3D::new(1.0, 2.0, 3.0); let v = Vector3D::new(1.0, 2.0, 3.0); let a = 42.0;

// Addition let w = u + v; // Substraction let w = u - v; // Negation let w = -u; // Cross product let w = u ^ v;

// Multiplication let w = a * u; let w = v * a; // Division let w = u / a;

// Dot product let b = u * v; ```
